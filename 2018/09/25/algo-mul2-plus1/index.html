算法笔记：乘2加1
发布于：2018-09-25 10:54:08 标签：/ javascript / algorithm / 访问：

问题描述

两个数A和B，其中A<B。只能对A进行两种操作：A=A+1 或 A=A*2，求A到B的最少操作次数。

解决思路

先判断符号位是否相等，如果不等，就一直A=A+1。然后判断二进制长度是否相等，如果不等，表明数值相差较大，可以尝试A=A*2。如果长度相等，表明数值非常接近，直接B-A。
